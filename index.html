About the Project

Welcome to Funds For All, a Web3 platform where anyone can create a funding poll for any cause — whether it's to stop wars, prevent starvation, build a house, or even fund a new robotic arm.

Users connect via MetaMask or any wallet, and vote for the cause they believe in. Each vote sends funds to the shared pool.

Once voting ends, the cause with the most votes wins and receives the full pool of funds.

A public chat is available, so the community can express thoughts, debate, or support each other’s causes.
